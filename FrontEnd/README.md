# SmartBite AI 🍽️

> **Good Food, Smart Choice!** — AI-powered food ordering with voice assistant.

## Features
- 🤖 AI food recommendations (Spicy / Healthy / Sweet / Fast Food)
- 🎤 Voice Assistant powered by speech recognition
- 🛒 Cart & checkout with UPI/COD payment
- 👤 User auth (login/signup)
- 📦 Order history & profile
- 🌙 Premium dark theme with orange accents
- 📱 Fully responsive

## Stack
- **Frontend**: React + Vite
- **Backend**: Django REST API
- **Voice**: Web Speech API + react-speech-recognition
- **Payment**: Razorpay

## Setup
```bash
cd FrontEnd
npm install
npm run dev
```

Backend runs on `http://localhost:3000`
