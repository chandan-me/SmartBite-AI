# 🍔 SMARTBITE AI

SMARTBITE AI is an AI-powered food ordering web application built with React, Django, Groq AI, and Razorpay.

## Features

* 🛒 Food Ordering
* 🎤 Voice Assistant Navigation
* 💳 Razorpay Payments
* 📦 Cash on Delivery (COD)
* 📜 Order History
* 👤 User Profile Management

## Tech Stack

* Frontend: React + Vite
* Backend: Django
* Database: SQLite
* AI: Groq API
* Payments: Razorpay

## Run Locally

### Backend

```bash
cd Backend
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver 3000
```

### Frontend

```bash
cd FrontEnd
npm install
npm run dev
```
